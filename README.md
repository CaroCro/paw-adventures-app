# PawAdventures

PawAdventures is a responsive front-end React project for a wildlife tours platform. It was designed as a portfolio project to showcase reusable components, interactive UI, modal-based tour details, filtering functionality, and form validation.

## Live Demo

[View the live site](https://paw-adventures-app.netlify.app)

## Features

- Responsive single-page layout
- Section-based navigation
- Interactive tour cards with modal detail views
- Dynamic filtering by destination
- Booking form with controlled inputs and validation
- Smooth scrolling navigation
- External social links

## Built With

- React
- JavaScript
- CSS
- Netlify
